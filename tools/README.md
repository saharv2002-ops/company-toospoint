# Tools

Internal tools and small apps that support the consulting practice.

| Tool | Purpose |
|---|---|
| [`rfp-search/`](./rfp-search) | Search engine for finding active RFPs / RFQs to bid on |

## RFP Search

- `rfq_search.py` — main script
- `run.sh` — convenience launcher
- `templates/index.html`, `templates/rfq_index.html` — Flask / Jinja templates for the web UI

Run with `./run.sh` from inside the `rfp-search/` directory.
