# Prospects

Active leads that haven't signed a contract yet. Once a contract is
executed, **move the prospect folder into `clients/`**.

## Standard prospect folder layout

Same shape as a client folder:

```
<prospect-name>/
├── README.md             # Who they are, what they want, status, contacts
├── proposals/            # Draft proposals, even pre-send
├── notes/                # Call notes, email threads, internal thinking
└── deck/                 # If we built a tailored pitch deck
```

Drop a `README.md` at the root of each prospect folder with at minimum:

- Primary contact + role
- What they do (business model, size, market)
- What they're asking for
- What the next step is
- Date of last contact
